# JouleUWP

Unofficial Windows 10 UWP App (phone, desktop, hololens, xbox) for ChefSteps Joule Sous-vide device.
This is only my fork of [Mitch Capper' JouleUWP](https://github.com/mitchcapper/JouleUWP) app.
